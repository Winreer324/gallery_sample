## Getting Started

generation icons

```dart
flutter pub run flutter_launcher_icons:main
```

start code gen:

```dart
- flutter packages pub run build_runner watch
```

or

```dart
- flutter packages pub run build_runner build
```

delete conflict:

```dart
- flutter packages pub run build_runner watch
```
or

```dart
- flutter packages pub run build_runner build --delete-conflicting-outputs
```
